# Projeto Validação

Este projeto foi desenvolvido para demonstrar um sistema simples de validação de dados em formulários.

## 📌 Sobre o projeto

O objetivo deste projeto é validar informações inseridas pelo usuário em um formulário, garantindo que os dados estejam corretos antes de serem enviados.

## ⚙️ Funcionalidades

* Validação de campos obrigatórios
* Verificação de formato de e-mail
* Mensagens de erro para dados inválidos
* Interface simples e fácil de usar

## 💻 Tecnologias utilizadas

* HTML
* CSS
* JavaScript

## 📂 Estrutura do projeto

* `index.html` – Estrutura da página
* `style.css` – Estilização da página
* `script.js` – Lógica de validação do formulário

## 🎯 Objetivo

Este projeto foi criado para fins de aprendizado em desenvolvimento web, com foco em validação de formulários no front-end.
